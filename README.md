## 📌 About The Project

**Widget World** is a community-driven web application designed to streamline Flutter mobile app development. Developers can browse a rich library of pre-built Flutter widgets, copy code instantly, publish their own widgets, and clone full application templates — all for free.

The platform eliminates the need for costly subscriptions by offering a robust, open-source alternative that empowers developers to build faster and smarter.

## 🏗️ System Architecture

The platform follows a clean **Client → API → Database** architecture:

```
User / Admin
    │
    ▼
Flutter Web Frontend
    │
    ▼
RESTful API (Node.js)
    │
    ▼
MySQL Database
```

**Core flows:**
- Users browse widgets → select → copy code into their project
- Users can publish widgets → admin reviews → approved widgets go live
- Users can browse & clone app templates directly
